# StockFlow

Sistema SaaS de gestão de estoque multiempresa.

## Tecnologias

Next.js
Prisma
PostgreSQL
Supabase
Tailwind
Recharts

## Funcionalidades

- Gestão de produtos
- Controle de estoque
- Movimentações
- Importação de NF-e
- Scanner de código de barras
- Sistema de solicitações
- Convites de usuários
- Alertas de estoque baixo
- Dashboard analítico
